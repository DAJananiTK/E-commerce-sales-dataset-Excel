# E-Commerce Store Data Analysis

## Overview
This project analyzes an e-commerce retail dataset containing customer, product, store, and sales transaction data across multiple U.S. regions.
The goal is to understand sales performance, customer behavior, product demand, and store efficiency using Excel-based analysis.

## Objectives

- Analyze overall sales and revenue performance
- Identify top customers, products, and stores
- Understand customer segmentation (loyalty, gender, region)
- Evaluate product category performance
- Study regional and store efficiency
- Analyze payment method usage and discount impact

## Dataset Description

The dataset includes the following tables:

- Customer Data
- Product Data
- Store Data
- Sales Data

## Tools Used

- Microsoft Excel
- Power Query
- Pivot Tables
- Excel Charts & Dashboards

- Revenue analysis by region, store, and product
- Customer segmentation analysis
- Payment method analysis
- Discount impact analysis
- Dashboard creation using slicers

## Key Insights

- Platinum customers generate the highest revenue
- Sports category is the top-performing category
- Online stores generate more revenue than physical stores
- Store 8 is the best-performing store
- West region shows lower performance
- PayPal is the most used payment method

## Conclusion

This project provides useful business insights that can help improve sales, optimize inventory, and enhance customer targeting strategies.
